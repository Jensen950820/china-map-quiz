# 中国互动地图 🇨🇳

交互式学习中国地理——从全国到各省，从地级市到区县。

## 快速开始

双击 `china/map.html` 即可在浏览器中打开全国地图。

## 功能

- **全国地图**：34个省级行政区答题，答对后跳转各省详情
- **省级地图**（33个）：每个省含地级市答题 + 区/县答题
- **双模式**：答题模式（猜名称）/ 浏览模式（直接跳转）
- **即时反馈**：答对绿色、答错红色，自动标注名称
- **完全离线**：所有地理数据内嵌，无需联网

## 目录结构

```
china/map.html      ← 入口：全国地图
{省名}/
  map.html          ← 各省地图
  gd_data.js        ← 行政区边界数据（GeoJSON）
  gd_counties.js    ← 县级边界数据
```
